# Sberbank Russian Housing Market Case

Проект по анализу данных и построению модели для предсказания стоимости недвижимости на основе данных о квартирах, районах и макроэкономических показателях.

В качестве целевой переменной используется price_doc — цена продажи объекта недвижимости.

## Что делает проект

Проект решает задачу регрессии: по характеристикам квартиры, района и экономической ситуации предсказывается стоимость объекта недвижимости.

Основной пайплайн:

1. Загрузка данных из train.csv и macro.csv.
2. Объединение данных по дате сделки timestamp.
3. Очистка аномальных значений.
4. Заполнение пропусков.
5. Удаление сильно коррелирующих и низковариативных признаков.
6. Генерация новых признаков.
7. Кодирование категориальных признаков.
8. Удаление выбросов по целевой переменной.
9. Обучение модели Lasso.
10. Подбор гиперпараметра alpha через GridSearchCV.
11. Оценка качества модели через TimeSeriesSplit и метрику RMSE.

## Структура проекта

text sberbank_case_1/ ├── main.py ├── preprocessor.py ├── train.csv ├── macro.csv └── data_dictionary.txt 

## Описание файлов

### main.py

Главный файл проекта.

В нём выполняется полный ML-пайплайн:

- загрузка данных;
- объединение основного датасета с макроэкономическими признаками;
- запуск препроцессинга;
- удаление выбросов по цене;
- разделение данных на признаки X и целевую переменную y;
- обучение модели Lasso;
- подбор параметра регуляризации;
- оценка качества на временных фолдах.

### preprocessor.py

Файл с классом Preprocessor, который отвечает за подготовку данных.

Основные методы:

- basic_clean() — очищает аномальные значения в признаках;
- fill_num_median() — заполняет пропуски в числовых признаках медианой;
- drop_high_corr() — удаляет сильно коррелирующие признаки;
- drop_low_variance() — удаляет признаки с низкой дисперсией;
- add_time_parts() — добавляет признаки года, месяца и квартала;
- feature_engineering() — создаёт новые признаки;
- drop_columns() — удаляет выбранные признаки;
- encode_cats() — кодирует категориальные признаки;
- transform() — возвращает обработанный датафрейм.

### train.csv

Основной датасет с информацией о сделках недвижимости.

Содержит:

- характеристики квартиры;
- данные о районе;
- инфраструктурные признаки;
- целевую переменную price_doc.

### macro.csv

Датасет с макроэкономическими показателями по датам.

Содержит признаки, связанные с экономикой: курс валют, инфляция, ВВП, ипотечные ставки, доходы населения и другие показатели.

### data_dictionary.txt

Описание признаков из train.csv и macro.csv.

## Используемая модель

В проекте используется модель линейной регрессии с L1-регуляризацией:

text Lasso Regression 

Lasso выбрана как базовая интерпретируемая модель, которая умеет занулять слабые признаки и снижать переобучение за счёт регуляризации.

## Оценка качества

Для оценки качества используется метрика:

text RMSE 

RMSE показывает среднюю ошибку модели в денежных единицах целевой переменной.

Для валидации используется:

text TimeSeriesSplit 

Такой способ разбиения выбран потому, что данные связаны со временем, и обычное случайное разбиение может дать некорректную оценку качества.
